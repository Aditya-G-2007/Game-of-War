# 🃏 Game of War - Guide

A fast-paced, browser-based card game built with **Vanilla JavaScript** and the **Deck of Cards API**. Test your luck against the computer!

## 🕹️ How to Play
1. Click **"New Deck"** to fetch a fresh, shuffled deck of 52 cards.
2. Click **"Draw Card"** to pull two cards—one for the Computer and one for You.
3. The player with the higher card value wins the round!
   - *Card Power:* Ace (High) > King > Queen > Jack > 10... down to 2 (Low).
4. The game ends when the deck is empty. The player with the most points wins the War!

## ✨ Features
- **Real-time API Integration:** Fetches live card data and images from [Deck of Cards API](https://deckofcardsapi.com).
- **Dynamic Scoring:** Automatically tracks scores and announces the winner.
- **Responsive Design:** Styled with custom Google Fonts (`Rye` and `Geom`) for a classic "Western" card game feel.
- **Async/Await:** Uses modern JavaScript for smooth, non-blocking API calls.

## 🛠️ Technologies Used
